### 문제 설명

<p>역사 속으로 사라져 버린 동국대학교의 명물 <strong>피막치</strong>를 아는가?</p>

<p>피막치는 피자, 막국수, 치킨의 조합을 뜻한다.</p>

<p>현재 더 이상 피막치를 만드는 곳이 없어, 이를 먹고 싶은 학생들은 세 가지를 따로 시켜 먹어야 한다.</p>

<p>피막치를 너무 좋아하는 해환이는 친구들과 매일 피자, 막국수, 치킨을 시켜 먹는다. 여러 번의 시행착오 끝에... 마침내!!</p>

<p>피막치의 황금 비율을 알아내었다. 바로 <strong>피막치수</strong>라고 하는 마법의 숫자 $X$이다.</p>

<p>피자의 개수를 $p$, 막국수의 개수를 $m$, 치킨의 개수를 $c$라고 할 때, $(p+m)\cdot(m+c)$ 의 값이 $X$에 가장 가까울수록 맛있는 피막치이다.</p>

<p>하지만 각 메뉴는 주문 가능한 <strong>최대 개수</strong>가 있다.</p>

<p>피자는 $1$개 이상 $P$개 이하, 막국수는 $1$개 이상 $M$개 이하, 치킨은 $1$개 이상 $C$개 이하 중에서 고를 수 있다.</p>

<p>모든 조합 중 $|(p+m)\cdot(m+c)-X|$ 의 <strong>최솟값</strong>을 구하여라.</p>

### 입력 

 <p>첫 번째 줄에 세 정수 $P, M, C$가 주어진다.</p>

<p>두 번째 줄에 정수 $X$가 주어진다.</p>

### 출력 

 <p>조건을 만족하는 최솟값을 정수 하나로 출력한다.</p>
