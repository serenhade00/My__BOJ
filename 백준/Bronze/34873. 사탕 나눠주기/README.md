### 문제 설명

<p>오늘 여러분은 지금 가지고 있는 사탕을 친구와 나눠 먹기로 했습니다.</p>

<p>여러분은 총 $2\times N$개의 사탕을 가지고 있으며, $i$번째 사탕은 맛의 종류를 나타내는 수 $a_i$를 가지고 있습니다.</p>

<p>여러분은 가지고 있는 사탕 중 $N$개를 친구에게 나눠주려고 합니다.</p>

<p>그런데, 같은 맛의 사탕을 여러 개 먹으면 쉽게 질리기 때문에, 여러분과 친구 모두 같은 종류의 사탕을 $2$개 가지고 있는 것을 좋아하지 않습니다.</p>

<p>과연 여러분과 친구 모두 서로 다른 종류의 사탕 $N$개를 가지고 있도록 친구에게 $N$개의 사탕을 줄 수 있을까요?</p>

### 입력 

 <p>첫 번째 줄에 양의 정수 $N$이 주어집니다.</p>

<p>두 번째 줄에 사탕의 종류를 나타내는 $2 \times N$개의 정수 $a_1,a_2,\ldots,a_{2N}$이 공백으로 구분되어 주어집니다.</p>

### 출력 

 <p>여러분과 친구 모두 서로 다른 종류의 사탕 $N$개를 가지고 있도록 할 수 있다면 "<code>Yes</code>"를, 아니면 "<code>No</code>"를 한 줄에 출력합니다.</p>
